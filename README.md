Loan-Approval-Prediction-Analysis-Dashboard
    This project analyzes loan approval data to identify key factors influencing approval decisions such as income, education, credit history, and loan amount.
Tools & Technologies
	•	Excel (Data Cleaning & Analysis)
	•	Power BI (Dashboard & Visualization)
	•	DAX (Data Analysis Expressions) for calculations.
  • Power Query for data cleaning.
  

🔄 Workflow

Data collection → Data cleaning → Feature analysis → Visualization → Insight generation

🔍 Key Insights
	•	Credit history is a major factor in loan approval
	•	Higher income increases approval chances
	•	Graduates show better approval rates
	•	Loan amount impacts approval probability

📈 Outcome

Developed an interactive dashboard to help financial institutions make informed loan approval decisions and reduce risk.
